### 주요 사업 분야

씨엠엔텍은 **초음파 유량계(Ultrasonic Flowmeter)** 및 **음파 수위계(Levelmeter)** 전문 제조기업입니다.

#### 핵심 제품군

1. **다회선 초음파 유량계** - 국내 최초 개발, 선구자 기업
2. **습식/건식 초음파 유량계**
3. **음파 수위계** - 고정밀
4. **압력전송기**
5. **전자유량계** (일체형, 분리형, 웨이퍼형, 클램프형)
6. **초음파 열량계**

#### 공급 실적

- **3,500대 이상** 국내외 시장 공급 (1995년~현재)
- 주요 고객: 수자원공사, 지자체 상하수도 사업소, 산업체

### 기업 비전 및 전략

> **"스마트 계측기 + IT 기술 융합 → 지능화된 수자원 관리 시스템(IWMS)"**

- 정밀 측정을 위한 스마트 계측기 개발
- 체계적이고 효율적인 수자원 관리 토탈 솔루션 제공
- 해외 네트워크를 통한 수출 확대 (다회선 초음파 유량계, 배터리 초음파 수도미터, 초음파 열량계)

### 현재 과제 및 AI 도입 필요성

#### 1. 견적 업무의 복잡성

**문제점**:
- 제품 라인업이 다양 (10개 이상 제품군)
- 고객별 맞춤형 사양 (설치 환경, 배관 크기, 유량 범위 등)
- 견적 작성 시간 평균 **2시간** 소요
- 오류 발생 시 재작성 필요 (20% 확률)

**도입 효과**:
- HEPHAITOS I/O 견적 자동화 → 평균 **24분**으로 단축 (80% 감소)
- AI 기반 오류 검증 → 오류율 **90% 감소**

#### 2. 생산 관리의 비효율

**문제점**:
- 주문 → 생산 → 출하 프로세스 수작업
- 재고 관리 실시간 파악 어려움
- 납기 지연 발생 (10% 확률)

**도입 효과**:
- 스마트공장 연동 → 실시간 생산 현황 파악
- 자동 재고 알림 → 결품 방지

#### 3. 품질 검사의 일관성 부족

**문제점**:
- 수동 검사에 의존 (사람마다 기준 차이)
- 불량률 2-3% 수준

**도입 효과**:
- AI 비전 검사 → 불량률 **0.5% 이하**

## HEPHAITOS I/O 솔루션 개요

### 개념

**HEPHAITOS I/O**는 제조업 특화 AI 자동화 플랫폼으로, 견적 자동화, 생산관리 연동, 품질검사 AI를 통합 제공합니다.

> **"Replit for Manufacturing"** - 코딩 없이 자연어로 업무 자동화

### 핵심 모듈

#### 1. 견적 자동화 모듈 (Quote AI)

| 기능 | 설명 |
|-----|------|
| **자연어 견적 생성** | "DN100 습식 유량계 10대 견적서" → 자동 생성 |
| **과거 견적 학습** | 기존 견적서 PDF/Excel 업로드 → AI 학습 |
| **실시간 가격 연동** | 원자재 가격, 환율 자동 반영 |
| **다국어 견적서** | 한국어, 영어, 일본어 자동 번역 |
| **견적서 포맷** | Excel, PDF, 한글 자동 변환 |

**기술 스택**:
- Claude 4 (Anthropic) - 자연어 이해
- Vercel AI SDK - 스트리밍 응답
- Supabase - 견적 데이터베이스

#### 2. 생산관리 연동 모듈 (MES Connect)

| 기능 | 설명 |
|-----|------|
| **주문서 자동 생성** | 견적 승인 → 생산 주문서 자동 생성 |
| **실시간 진행 현황** | 생산 → 검사 → 출하 단계별 추적 |
| **재고 자동 알림** | 부품 재고 부족 시 알림 + 자동 발주 |
| **납기 예측 AI** | 과거 데이터 기반 납기일 예측 |

**연동 대상**:
- ERP 시스템 (더존, SAP, 자체 개발)
- 스마트공장 MES
- 재고관리 시스템

#### 3. 품질검사 AI 모듈 (Vision QC)

| 기능 | 설명 |
|-----|------|
| **외관 검사** | 카메라 촬영 → AI 불량 판정 |
| **성능 검사 자동화** | 유량계 성능 시험 데이터 자동 분석 |
| **검사 리포트 생성** | 검사 결과 PDF 자동 생성 |

**기술 스택**:
- TensorFlow Lite - 경량 AI 모델
- OpenCV - 이미지 처리
