### Q1. HEPHAITOS는 무료인가요?

**A**: 기본 기능은 무료입니다. 다음 기능을 무료로 사용할 수 있습니다:
- ✅ 전략 생성 (최대 3개)
- ✅ 백테스팅 (월 10회)
- ✅ Paper Trading (무제한)
- ✅ 기본 지표 사용

**Pro 플랜** (월 29,000원):
- ✅ 무제한 전략 생성
- ✅ 무제한 백테스팅
- ✅ 고급 지표 (200+ 종류)
- ✅ 실전 거래 연동
- ✅ AI 멘토 코칭
- ✅ 우선 지원

### Q2. HEPHAITOS가 투자 조언을 해주나요?

**A**: **아니요**. HEPHAITOS는 교육 및 도구 제공 플랫폼입니다.

- ❌ 투자 조언 제공 안 함
- ❌ 수익 보장 안 함
- ❌ 특정 종목 추천 안 함
- ✅ 교육 목적의 도구 제공
- ✅ 사용자가 직접 전략 설계
- ✅ 백테스팅 및 시뮬레이션 지원

**모든 투자 결정은 본인 책임입니다.**

---

### Q3. 백테스팅 결과가 좋으면 실전에서도 성공하나요?

**A**: **아니요, 보장할 수 없습니다**.

백테스팅과 실전의 차이:
- **슬리피지**: 주문가와 실제 체결가의 차이
- **수수료**: 실제 거래 비용
- **시장 변화**: 과거와 현재 시장 환경이 다름
- **과최적화**: 과거 데이터에만 잘 맞는 전략

**권장사항**:
1. Paper Trading으로 충분히 테스트
2. 여러 기간/심볼로 백테스팅
3. 소액으로 실전 테스트
4. 리스크 관리 철저히

⚠️ **과거 성과는 미래 수익을 보장하지 않습니다.**

---

### Q4. 증권사 연동이 안 돼요.

**A**: 다음을 확인하세요:

**1. API 키 확인**
- API Key, Secret이 정확한지 확인
- 복사/붙여넣기 시 공백 없는지 확인

**2. 계좌번호 확인**
- 증권사 계좌번호가 올바른지 확인
- KIS: 8자리 계좌번호

**3. 증권사 서비스 상태**
- 증권사 API 서버가 정상인지 확인
- 점검 시간대 피하기

**4. IP 화이트리스트** (일부 증권사)
- 증권사 사이트에서 IP 주소 등록

**5. 권한 확인**
- API 키에 거래 권한이 있는지 확인

그래도 안 되면: support@ioblock.com으로 문의

### Q7. API 키가 유출되면 어떻게 되나요?

**A**: API 키가 유출되면 다음 위험이 있습니다:
- 타인이 계좌에 접근 가능
- 무단 거래 실행 가능
- 자산 손실 위험

**즉시 조치**:
1. 증권사 사이트에서 **API 키 폐기**
2. 새 API 키 발급
3. 비밀번호 변경
4. 계좌 활동 내역 확인

**예방 수칙**:
- API 키를 GitHub에 올리지 말기
- 환경 변수 (.env) 사용
- 정기적으로 키 교체
- 최소 권한만 부여

### Q8. 손절/익절 설정은 어떻게 하나요?

**A**: 전략 설정에서 리스크 관리 옵션을 사용하세요.

**예시**:
```
리스크 관리 설정
- 손절 (Stop Loss): -5%
- 익절 (Take Profit): +10%
- 최대 포지션 크기: 10%
```

**권장 설정**:

| 투자 성향 | 손절 | 익절 | 포지션 크기 |
|----------|------|------|-------------|
| 보수적 | -3% | +5% | 5% |
| 중립적 | -5% | +10% | 10% |
| 공격적 | -8% | +15% | 20% |

⚠️ **중요**: 손절은 필수입니다! 큰 손실을 방지합니다.

### Q9. 백테스팅이 너무 오래 걸려요.

**A**: 백테스팅 시간은 다음 요인에 따라 다릅니다:

| 요인 | 영향 | 권장 |
|------|------|------|
| **기간** | 길수록 느림 | 1년 이하 |
| **심볼** | 데이터 많을수록 느림 | 주요 종목 |
| **복잡도** | 조건 많을수록 느림 | 단순하게 |
| **큐 대기** | 대기자 많으면 느림 | 한가한 시간 |

**예상 시간**:
- 1개월 백테스팅: ~30초
- 1년 백테스팅: ~2분
- 5년 백테스팅: ~10분

**팁**: Pro 플랜 사용자는 우선 처리됩니다.

### Q10. 실전 거래 시 주의사항은?

**A**: 다음 사항을 반드시 지키세요:

**1. 소액으로 시작**
- 전체 자산의 5% 이하로 시작
- 익숙해지면 점진적으로 증가

**2. 리스크 관리**
- 손절 설정 필수
- 포지션 크기 제한
- 분산 투자

**3. 감정 통제**
- 공포와 탐욕 경계
- 계획대로 실행
- 충동적 거래 금지

**4. 지속적 모니터링**
- 정기적으로 성과 확인
- 전략 조정 필요 시 즉시 대응

**5. 법률 준수**
- 세금 신고 의무 확인
- 불법 거래 금지
- 내부자 거래 금지

⚠️ **원금 손실 위험이 있습니다. 투자는 신중하게!**
