# HEPHAITOS 카피 전략 (Pain-Based)

## 🎯 타겟 페르소나: 민수 (32세, IT 직장인)

### Pain Points

```
❌ "유튜브 따라했는데... 또 손절"
❌ "퀀트 공부하려니 Python부터?"
❌ "매달 9,900원 내는데 이번 달 안 썼어"
❌ "Nancy Pelosi 따라하고 싶은데 어떻게?"
❌ "AI 전략? 나는 코딩 못하는데..."
```

---

## 💡 핵심 메시지

### 1. 메인 캐치프레이즈

**Before (기존)**:
```
"쓴 만큼만 내는 투자 교육 플랫폼"
```

**After (Pain 기반)**:
```
"유튜브 따라하기 지쳤다면,
 이제 AI가 전략 짜드립니다"
```

또는

```
"Nancy Pelosi처럼 투자하고 싶은데
 코딩은 1도 모르는 당신에게"
```

### 2. 서브 캐치프레이즈

**Before**:
```
"💎 크레딧 기반 'Replit for Trading'"
```

**After**:
```
"월 구독 부담 NO, 쓴 만큼만 크레딧 충전
 안 쓰면 0원, 쓰면 1크레딧부터"
```

## 🔥 Pain-Based 카피 시트

### Landing Page 히어로 섹션

```
[큰 헤드라인]
"유튜버 따라하다 -15% 박은 당신,
 이제 AI한테 맡기세요"

[서브 헤드라인]
셀럽 포트폴리오 무료로 보고 → AI가 전략 자동 생성 → 코딩 없이 백테스트
월 구독 없이, 쓴 만큼만 크레딧 충전

[CTA 버튼]
"Nancy Pelosi 포트폴리오 무료로 보기 →"
```

### Feature 1: COPY (셀럽 미러링)

**Before**:
```
셀럽 포트폴리오 1클릭 미러링
```

**After**:
```
"Nancy Pelosi +73% 수익 비결,
 이제 공짜로 따라해보세요"

무료 | 0 크레딧 | 회원가입만 하면 끝
```

### Feature 2: LEARN (AI 튜터 + 코칭)

**Before**:
```
AI + 인간 멘토 하이브리드 코칭
```

**After**:
```
"왜 샀는지도 모르고 샀다면?
 AI 튜터가 1크레딧에 설명해드립니다"

AI 질문 1크레딧 | 전문가 코칭 20크레딧 (30분)
유튜브는 일방적, HEPHAITOS는 맞춤형
```

### Feature 3: BUILD (AI 전략 생성)

**Before**:
```
자연어 → Python 전략 → 24시간 자동매매 봇
```

**After**:
```
"코딩 1도 모르는데 퀀트?
 이제 한국어로 말하면 AI가 전략 짭니다"

10 크레딧에 전략 생성 + 3 크레딧에 1년 백테스트
Python 몰라도 OK | 자연어면 충분합니다
```

## 💎 크레딧 시스템 카피

### 기존 월 구독의 Pain

```
❌ 스타터 ₩9,900/월 → "이번 달 안 썼는데 왜 내?"
❌ 프로 ₩29,900/월 → "가끔만 쓰는데 비싸"
```

### 크레딧의 Solution

```
✅ 스타터 100크레딧 ₩9,900
   → "안 쓰면 다음 달에 써도 됩니다"

✅ 프로 1,000크레딧 ₩69,000
   → "대량 구매 시 크레딧당 ₩60으로 저렴"

무료 체험 50 크레딧 증정
친구 추천 시 30 크레딧 추가
```

## 🚀 랜딩 페이지 구성 (Pain → Agitate → Solve)

### Section 1: Pain (공감)

```
"이런 적 없으신가요?

✓ 유튜브 따라했는데 주가는 이미 올라간 후
✓ Python 공부하려다 포기한 퀀트 투자
✓ 매달 ₩9,900 내는데 이번 달 안 쓴 구독
✓ Nancy Pelosi 따라하고 싶은데 방법 모름
✓ AI 전략? 코딩 몰라서 포기"
```

### Section 2: Agitate (강화)

```
"그동안 당신이 놓친 것들

• Nancy Pelosi는 작년에만 +73% 수익
• 퀀트 트레이더는 감정 없이 데이터로만 매매
• ChatGPT로 코드 짜는 시대, 아직도 수동 매매?

문제는 '정보 부족'이 아니라
'실행 방법을 몰라서' 입니다"
```

### Section 3: Solve (해결)

```
"HEPHAITOS가 답입니다

1단계: COPY
   → Nancy Pelosi 포트폴리오 공짜로 보고 따라하기

2단계: LEARN
   → AI 튜터에게 1크레딧에 "왜 샀는지" 물어보기

3단계: BUILD
   → 한국어로 전략 설명하면 AI가 코드로 변환

코딩 몰라도 OK | 월 구독 NO | 쓴 만큼만 충전
```
