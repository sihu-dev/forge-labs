## Executive Summary

**HEPHAITOS**는 개인 투자자가 코딩 없이 자연어만으로 자신만의 AI 트레이딩 봇을 만들 수 있는 **투자 자동화 플랫폼**입니다.

Replit이 "코딩 모르는 사람도 앱을 만들 수 있게" 한 것처럼, HEPHAITOS는 **"투자 전문가가 아니어도 AI 트레이딩 시스템을 구축"**할 수 있게 합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    HEPHAITOS = Replit for Trading               │
├─────────────────────────────────────────────────────────────────┤
│  Replit: "Make me a todo app"  →  React 코드 자동 생성          │
│  HEPHAITOS: "워렌 버핏처럼 투자해줘" → Python 전략 자동 생성     │

└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Trading Agent

> **스스로 판단하고 실행하는 AI 에이전트**

```python
# HEPHAITOS AI Agent 예시
class TradingAgent:
    """
    사용자의 자연어 지시를 이해하고
    스스로 시장을 분석하여 매매를 실행하는 AI
    """

    def __init__(self, strategy: str, broker: UnifiedBroker):
        self.strategy = strategy  # "워렌 버핏 스타일 가치투자"
        self.broker = broker
        self.ai = ClaudeAPI()

    async def analyze_market(self):
        """실시간 시장 분석"""
        data = await self.get_market_data()
        analysis = await self.ai.analyze(data, self.strategy)
        return analysis

    async def make_decision(self):
        """AI 기반 매매 결정"""
        analysis = await self.analyze_market()
        if analysis.signal == "BUY":
            await self.broker.buy(analysis.symbol, analysis.quantity)
        elif analysis.signal == "SELL":
            await self.broker.sell(analysis.symbol, analysis.quantity)

    async def run_24_7(self):
        """24시간 자동 실행"""
        while True:
            await self.make_decision()
            await asyncio.sleep(self.interval)
```

**Agent 특징**:
| 특징 | 설명 |
|------|------|
| **자율성** | 사용자 개입 없이 24시간 동작 |
| **학습** | 매매 결과 피드백으로 지속 개선 |
| **설명** | 왜 그 결정을 했는지 설명 가능 |
| **안전** | 손절/이익실현 자동, 최대 손실 제한 |

## 5. 기술적 차별점

### 5.1 UnifiedBroker API

> **3분 내 증권사 연동** (기존 2-3일 → 3분)

```typescript
// 모든 증권사를 하나의 인터페이스로
interface UnifiedBroker {
  connect(credentials: BrokerCredentials): Promise<ConnectionResult>;
  getBalance(): Promise<Balance>;
  getHoldings(): Promise<Holding[]>;
  buy(symbol: string, qty: number, price?: number): Promise<OrderResult>;
  sell(symbol: string, qty: number, price?: number): Promise<OrderResult>;
  subscribePrice(symbol: string, callback: PriceCallback): void;
}

// 증권사별 어댑터 (내부 구현)
class KISAdapter implements UnifiedBroker { /* 한국투자증권 */ }
class KiwoomAdapter implements UnifiedBroker { /* 키움증권 */ }
class AlpacaAdapter implements UnifiedBroker { /* 미국 Alpaca */ }
```

**지원 증권사**:
| 국가 | 증권사 | 연동 방식 |
|------|--------|----------|
| 🇰🇷 한국 | 한국투자증권 | REST API |
| 🇰🇷 한국 | 키움증권 | Open API |
| 🇰🇷 한국 | 삼성/미래에셋 | 개발 중 |
| 🇺🇸 미국 | Alpaca | REST API |
| 🇺🇸 미국 | Interactive Brokers | 개발 중 |
