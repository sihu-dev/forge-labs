# HEPHAITOS 트레이딩 플랫폼 규칙

## 법률 준수 (CRITICAL)
- 투자 조언 절대 금지
- "~하세요" 권유형 표현 금지
- 모든 트레이딩 UI에 면책조항 필수

## 허용 표현
- "~할 수 있습니다" (설명형)
- "이 전략은 ~한 특징이 있습니다"
- "과거 성과는 미래를 보장하지 않습니다"

## 금지 표현
- "수익 보장", "확실한 수익"
- 구체적 종목 추천
- "이 전략을 사용하세요"

## 디자인
- Dark Mode Only (#0D0D0F)
- Primary Color: #5E6AD2 (Linear Purple)
- Glass Morphism 패턴
