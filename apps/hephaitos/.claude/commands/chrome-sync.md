# Chrome Claude 컨텍스트 동기화

## 현재 세션 정보
- **프로젝트**: HEPHAITOS (AI 트레이딩 교육 플랫폼)
- **서버**: http://localhost:3000
- **모드**: Mock (Supabase 미사용)

## 핵심 페이지

| 경로 | 설명 | 테스트 포인트 |
|------|------|--------------|
| `/` | 랜딩 페이지 | Hero, 제품 카드, CTA |
| `/dashboard` | 대시보드 | 포트폴리오, 전략 목록 |
| `/dashboard/strategy-builder` | 전략 빌더 | No-Code 블록, 드래그앤드롭 |
| `/dashboard/backtest` | 백테스트 | 차트, 성과 지표 |
| `/auth/login` | 로그인 | 폼 유효성 |
| `/auth/signup` | 회원가입 | 폼 유효성 |

## Chrome Claude 역할

### 1. 시각적 QA
- UI 렌더링 확인
- 레이아웃 깨짐 감지
- 반응형 테스트

### 2. UX 검증
- 사용자 플로우 테스트
- 버튼/링크 동작 확인
- 에러 상태 확인

### 3. 컨텍스트 공유
Chrome Claude가 발견한 이슈를 Claude Code에 전달:
```
발견된 오류:
- 위치: /dashboard
- 증상: 버튼 클릭 시 무반응
- 스크린샷: [첨부]
```

## Claude Code 역할

### 1. 코드 수정
Chrome Claude 피드백 기반 즉시 수정

### 2. 서버 관리
- 개발 서버 실행/재시작
- 빌드 오류 해결

### 3. 테스트 자동화
- E2E 테스트 작성
- 단위 테스트 보강

## 협업 워크플로우

```
┌─────────────────┐     피드백      ┌─────────────────┐
│  Chrome Claude  │ ───────────────→ │   Claude Code   │
│  (브라우저 QA)  │                   │  (코드 수정)    │
└─────────────────┘ ←─────────────── └─────────────────┘
                      수정 완료 알림
```

## 현재 실행 중인 서버

```bash
# HEPHAITOS 개발 서버
pnpm --filter hephaitos dev  # 포트 3000
```

## 단축 명령

| 입력 | 동작 |
|------|------|
| `확인해줘 /dashboard` | 대시보드 페이지 QA |
| `빌더 테스트` | 전략 빌더 전체 테스트 |
| `오류 리포트` | 발견된 오류 정리 |
