# /strategy - AI 전략 생성

자연어로 트레이딩 전략을 생성하고 백테스팅합니다.

## 사용법

```
/strategy [자연어 설명]
```

## 예시

```
/strategy 이동평균선 20일선이 50일선을 상향 돌파하면 매수, 하향 돌파하면 매도

/strategy 워렌 버핏 스타일 가치투자 전략

/strategy RSI 30 이하에서 매수, 70 이상에서 매도

/strategy 볼린저 밴드 하단 터치 시 매수, 상단 터치 시 매도
```

## 작업 프로세스

1. **자연어 분석**
   - 사용자 입력에서 진입/청산 조건 추출
   - 기술 지표 파악
   - 리스크 관리 규칙 파악

2. **전략 코드 생성**
   - Python 백테스팅 코드 생성
   - Visual Builder 블록 조합 생성

3. **백테스팅 실행**
   - 과거 1년 데이터로 시뮬레이션
   - 성과 지표 계산 (수익률, Sharpe Ratio, MDD)

4. **결과 리포트**
   - 백테스팅 결과 요약
   - 차트 시각화
   - 면책조항 포함

## 주의사항

- ✅ 교육 목적의 전략 생성
- ❌ 투자 조언이 아님
- ❌ 과거 성과 ≠ 미래 보장

## 관련 파일

- `src/lib/agent/strategy-generator.ts` - AI 전략 생성
- `src/lib/backtest/backtester.ts` - 백테스팅 엔진
- `src/components/strategy-builder/` - Visual Builder

---

당신은 HEPHAITOS의 AI 전략 생성 전문가입니다.

사용자의 자연어 입력을 분석하여 실행 가능한 트레이딩 전략을 생성하세요.

**반드시 다음을 포함하세요:**
1. 전략 개요
2. 진입 조건
3. 청산 조건
4. 리스크 관리
5. 백테스팅 결과
6. 면책조항

**면책조항 필수:**
```
※ 본 전략은 교육 목적이며, 투자 조언이 아닙니다.
※ 백테스트 결과는 과거 데이터 기반이며, 미래 수익을 보장하지 않습니다.
※ 투자 결정은 본인 책임입니다.
```
