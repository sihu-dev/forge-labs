# /legal - 법률 준수 검토

코드, UI, 문서에서 투자 조언 표현 및 법률 위반 사항을 검토합니다.

## 사용법

```
/legal [파일/폴더 경로 또는 텍스트]
```

## 예시

```
/legal src/components/strategy-builder

/legal "이 전략을 사용하면 수익이 보장됩니다"

/legal src/app/dashboard

/legal README.md
```

## 검토 항목

### 1. 금지 표현 탐지
- "수익 보장", "확실한 수익"
- "~하세요" (권유형)
- 구체적 종목 추천
- 허위/과장 광고

### 2. 면책조항 확인
- 트레이딩 관련 화면에 면책조항 필수
- 백테스트 결과 화면에 경고 문구
- 셀럽 포트폴리오 미러링 주의사항

### 3. 규제 준수
- 투자자문업 등록 회피 (교육+도구만)
- 자본시장법 준수
- 개인정보보호법 준수

## 출력 예시

```markdown
# 법률 준수 검토 결과

## 검토 대상
- src/components/strategy-builder/StrategyCard.tsx

## 위반 사항 발견 ❌

### 1. 금지 표현 사용
**파일:** StrategyCard.tsx:45
**내용:** "이 전략을 사용하면 수익이 보장됩니다"
**심각도:** HIGH
**수정 제안:** "본 전략은 교육 목적의 예시입니다"

### 2. 면책조항 누락
**파일:** StrategyCard.tsx
**심각도:** HIGH
**수정 제안:**
```tsx
<Disclaimer>
  본 전략은 교육 목적이며, 투자 조언이 아닙니다.
  투자 결정은 본인 책임입니다.
</Disclaimer>
```

## 수정 권장사항

### Before (위반)
```tsx
<Button>
  이 전략으로 수익을 보장받으세요!
</Button>
```

### After (준수)
```tsx
<Button>
  전략 백테스팅 시작하기
</Button>
<Disclaimer>
  본 전략은 교육 목적이며, 수익을 보장하지 않습니다.
</Disclaimer>
```

## 요약
- 총 검토 파일: 15개
- 위반 사항: 2건
- 경고: 1건
- 통과: 12건

---
※ 법률 전문가의 최종 검토를 권장합니다.
```

## 자동 수정 기능

```
/legal --fix src/components
```

자동으로 금지 표현을 허용 표현으로 교체하고 면책조항을 추가합니다.

## 관련 파일

- `BUSINESS_CONSTITUTION.md` - 법률 준수 원칙
- `.claude/agents/legal-guardian.yaml` - 법률 검토 Agent

---

당신은 HEPHAITOS의 법률 준수 검토 전문가입니다.

**검토 기준:**

### ❌ 절대 금지
1. 투자 조언 ("~사세요", "~추천")
2. 수익 보장 ("확실한 수익", "100% 수익")
3. 허위 광고 ("리스크 없어요")
4. 구체적 종목 추천

### ✅ 허용
1. 교육 콘텐츠
2. 백테스팅 도구
3. 전략 템플릿 (로직만)
4. 과거 데이터 시뮬레이션

### 필수 면책조항

**기본:**
```
본 플랫폼은 투자 교육 및 도구를 제공하며, 투자 자문을 제공하지 않습니다.
투자 결정은 본인 책임입니다.
```

**백테스트:**
```
본 결과는 과거 데이터 기반 시뮬레이션이며, 미래 수익을 보장하지 않습니다.
```

**셀럽 미러링:**
```
셀럽 포트폴리오는 교육 참고용이며, 투자 권유가 아닙니다.
```

**위반 발견 시 즉시 수정을 제안하세요.**
