## 3. 솔루션 제안: HEPHAITOS I/O

### 핵심 가치 제안

```
┌─────────────────────────────────────────────────────────────────┐
│  "씨엠엔텍의 과거 300건 견적 데이터를 학습한 AI가                 │
│   입찰공고만 업로드하면 3분 내 견적서 초안 자동 생성"             │
│                                                                 │
│  Input:  나라장터 입찰공고 PDF                                   │
│  Output: 씨엠엔텍 양식 견적서 Excel (95% 완성도)                 │
│  Time:   3시간 → 30분 (AI 3분 + 검토 27분)                      │
└─────────────────────────────────────────────────────────────────┘
```

### 작동 방식 (3단계)

#### Step 1: 입찰공고 자동 분석 (AI, 2분)
```python
# HEPHAITOS I/O가 수행하는 작업
1. PDF OCR 텍스트 추출
2. 핵심 정보 파싱:
   - 유량계 유형 (초음파/전자/와류/...)
   - 수량 (예: 초음파유량계 DN100 × 15EA)
   - 설치 환경 (상수도/하수도/산업용수)
   - 납기일
   - 특수 요구사항 (방폭, IP등급, 통신규격 등)
3. 과거 유사 입찰 검색 (Vector DB)
```

#### Step 2: 견적서 초안 자동 생성 (AI, 1분)
```
AI 학습 데이터:
- 씨엠엔텍 과거 300건 견적서
- 낙찰/탈락 패턴 분석
- 경쟁사 가격 데이터 (나라장터 공개)

자동 생성 항목:
✅ 주품목 단가 (시장가 기준 ±10% 범위)
✅ 부자재 자동 산출 (설치환경 기반)
✅ 공사비 (설치 난이도 자동 계산)
✅ 제안서 초안 (과거 수주 케이스 기반)
```

#### Step 3: 담당자 검토 및 제출 (인간, 27분)
```
담당자 역할:
1. AI 생성 견적서 검토 (10분)
   - 단가 적정성 확인
   - 특수사항 반영 여부 확인
2. 세부 조정 (12분)
   - 영업 전략 반영 (공격적/안정적 가격)
   - 고객 특성 고려
3. 최종 승인 및 제출 (5분)
```
