> **조사 기간**: 2025-12-16
> **타겟**: 30-40대 직장인 재테크족 ("재테크 지영" 페르소나)

## 📊 핵심 발견

### 구독 서비스 이용 현황

**KB국민카드 설문 조사 결과** ([출처](https://blog.opensurvey.co.kr/trendreport/subscription-service-2025/)):
- 전체 평균 구독 개수: **월 5.8개**
- 30~39세: **32%가 7개 이상** 구독
- 25~29세: 평균 **7.3개** (43%가 7개 이상)
- 74%가 **두 개 이상 서비스** 동시 구독

**급성장 분야**:
1. **생성형 AI 구독**: 전년 대비 **299% 증가**
   - 10명 중 3명이 **7개월 이상 유지**
2. **생활·건강**: 59% 증가 (46만 → 74만 건)
3. **쇼핑·배달 멤버십**: 34%
4. **뉴스·매거진**: 32%

**시장 규모**: 2025년 국내 구독경제 **100조 원 돌파 예상**

## 🎯 실제 Pain Point (30-40대 재테크족)

### 1순위: "시간 없어요" ⏰

```
"직장 다니면서 주식 공부할 시간이 어디 있어요?"
"퇴근하면 애들 봐야 하는데..."
"주말에 유튜브 보는 게 고작"
```

**증거**:
- 평균 구독 개수 **5.8개** → 시간 절약 위해 구독 선호
- "월급쟁이부자들" 같은 직장인 특화 서비스 인기

### 2순위: "믿을 수 있나요?" 🤔

```
"AI가 주식 전략? 그거 진짜 먹혀요?"
"백테스팅이 뭔지도 모르겠고..."
"유튜버들은 뭘 믿고 따라해야 하는지..."
```

**증거**:
- 슈카월드 350만 구독자 → **신뢰할 수 있는 전문가** 찾음
- 삼프로TV "증권사 애널리스트" 강조 → **검증된 전문가** 선호
- 자동매매 프로그램 15만원 → **검증 비용 기꺼이 지불**

### 3순위: "손절/익절 타이밍 모름" 📉📈

```
"언제 팔아야 할지 모르겠어요"
"손절은 머리론 알지만 실행이..."
"고점에 사서 저점에 파는 게 제 특기"
```

**증거**:
- 자동매매 프로그램 수요 → **감정 개입 제거** 원함
- "손절/익절 자동" 기능 인기

### 4순위: "비싸요" 💸

```
"투자 자문 월 50만원? 너무 비싸요"
"유료 강의도 10만원 넘으면 망설여져요"
"구독료가 쌓이면 부담돼요"
```

**증거**:
- 크레딧 기반, 쓴 만큼만 → **합리적 소비** 선호
- 구독 7개 이상 → **피로감** 느낌
- 챗GPT 유료 구독 299% 증가 → 하지만 **ROI 명확**한 서비스만

### 5순위: "고수 따라하고 싶어요" 👥

```
"Nancy Pelosi 포트폴리오 궁금해요"
"워렌 버핏은 뭘 샀을까?"
"슈카월드에서 추천한 종목 관심 가요"
```

**증거**:
- 셀럽 투자 추적 서비스 (Unusual Whales) $999/월
- "신사임당" 채널 20억원 매각
- 삼프로TV 인터뷰 방식 인기

## 🎯 타겟 페르소나 (최종)

### "재테크 지영" (37세, 회사원)

**프로필**:
- 직장: 중견기업 마케팅팀 과장
- 연봉: 6,000만원
- 가족: 맞벌이 부부 + 초등학생 자녀 1명
- 투자 경험: 3년 (적금 → 주식 → ETF)
- 투자 원금: 5,000만원
- 목표: 연 10% 수익 (노후 준비)

**하루 일과**:
```
07:00 - 출근 준비 (아이 등교)
09:00 - 회사 업무
12:00 - 점심시간 (유튜브 쇼츠)
18:00 - 퇴근
19:00 - 저녁 + 육아
21:00 - 자유시간 (30분)
  → 삼프로TV 보거나
  → 주식 체크
22:00 - 취침
```

**Pain Point**:
1. "퇴근 후 30분밖에 못 봐요"
2. "AI 믿어도 되나요?"
3. "손절 타이밍 알려주면 좋겠어요"
4. "월 50만원 투자 자문은 너무 비싸요"
5. "Nancy Pelosi 따라하고 싶어요"

**구독 서비스 (7개)**:
1. 넷플릭스 (₩9,500)
2. 멜론 (₩10,900)
3. 쿠팡와우 (₩4,990)
4. 챗GPT Plus (₩20,000)
5. 삼프로TV 프리미엄 (₩30,000)
6. 네이버 프리미엄 (₩4,900)
7. **HEPHAITOS 크레딧** (₩10,000 월평균)

**의사결정 프로세스**:
```
1. 유튜브에서 "주식 AI" 검색
2. 슈카월드 영상 시청 (10분)
3. 댓글 확인 ("이거 진짜 먹혀요?")
4. 무료 체험 (50 크레딧)
5. Nancy Pelosi 따라하기 (0 크레딧)
6. AI 튜터 질문 3회 (3 크레딧)
7. 만족 → 10,000원 구매 (100 크레딧)
8. 월 1-2회 AI 전략 생성 (10 크레딧씩)
```
