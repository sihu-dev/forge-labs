# HEPHAITOS 사업계획서

> **버전**: 1.0.0
> **작성일**: 2025-12-19
> **상태**: 확정

---

## 1. Executive Summary

### 1.1 원라인 피치
```
"증권/코인 유튜버를 위한 White-Label AI 트레이딩 교육 플랫폼"
```

### 1.2 핵심 가치 제안
- **유튜버**: 기술 없이 유료 교육 플랫폼 즉시 운영
- **수강생**: 교육 수료 후 나만의 AI 트레이딩 에이전트 생성
- **HEPHAITOS**: 유튜버 네트워크로 CAC 제로 고객 획득

### 1.3 비즈니스 모델
```
B2B2C: HEPHAITOS → 유튜버(B2B) → 수강생(B2C)
```

## 2. 시장 분석

### 2.1 TAM/SAM/SOM

| 구분 | 규모 | 산정 근거 |
|------|------|----------|
| **TAM** | 50조원 | 글로벌 온라인 트레이딩 교육 시장 |
| **SAM** | 5,000억 | 한국 증권/코인 교육 시장 |
| **SOM** | 100억 | 유튜버 파트너십 기반 점유 (Year 3) |

### 2.2 타겟 세그먼트

#### 유튜버 (B2B)
| Tier | 구독자 | 특징 | 목표 |
|------|--------|------|------|
| Tier 1 | 50만+ | 메가 인플루언서 | 5개 |
| Tier 2 | 10-50만 | 매크로 인플루언서 | 30개 |
| Tier 3 | 1-10만 | 마이크로 인플루언서 | 65개 |

#### 수강생 (B2C)
- 주식/코인 입문자 (60%)
- 수동 트레이더 → 자동화 희망 (30%)
- 퀀트 입문자 (10%)

### 2.3 경쟁 분석

| 경쟁사 | 강점 | 약점 | 우리 차별점 |
|--------|------|------|------------|
| 클래스101 | 브랜드 인지도 | 도구 없음 | 교육+도구 통합 |
| TradingView | 차트 도구 | 교육 없음 | 유튜버 교육 채널 |
| 3Commas | 자동화 도구 | 교육/파트너십 없음 | B2B2C 모델 |

## 3. 제품/서비스

### 3.1 핵심 기능

```
┌─────────────────────────────────────────────────────────────────┐
│                    HEPHAITOS 플랫폼 구성                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [유튜버 대시보드]          [수강생 플랫폼]                      │
│  ├─ 화이트라벨 브랜딩       ├─ 강의 수강                        │
│  ├─ 커리큘럼 빌더           ├─ No-Code 전략 빌더                │
│  ├─ 수강생 관리             ├─ 백테스트 엔진                    │
│  ├─ 정산 대시보드           ├─ AI 에이전트 생성                 │
│  └─ 커뮤니티 관리           └─ 실계좌 연결 (BYOK)               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 수강생 여정

```
YouTube 시청 → 유튜버 채널 랜딩 → 수강 신청 (₩30만)
                                      ↓
┌─────────────────────────────────────────────────────────────────┐
│  STAGE 1: 수강 (4주)                                            │
│  Week 1: 기초 → Week 2: 분석 → Week 3: 전략 → Week 4: 자동화    │
└─────────────────────────────────────────────────────────────────┘
                                      ↓
┌─────────────────────────────────────────────────────────────────┐
│  STAGE 2: 에이전트 빌드                                         │
│  전략 조합 → 백테스트 → AI 개선 제안 → 에이전트 완성            │
└─────────────────────────────────────────────────────────────────┘
                                      ↓
┌─────────────────────────────────────────────────────────────────┐
│  STAGE 3: 실전 운용                                             │
│  계좌 연결 (BYOK) → 자동 매매 → 모니터링 → 성과 리포트          │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 핵심 기술

| 기술 | 설명 |
|------|------|
| **No-Code 전략 빌더** | 17개 블록 드래그앤드롭 |
| **백테스트 엔진** | 22개 성과 지표 계산 |
| **AI 개선 제안** | 전략 최적화 자동 추천 |
| **BYOK (Bring Your Own Key)** | 사용자 API 키로 거래소 연결 |
| **멀티 거래소** | Binance, Upbit, Bithumb, Coinbase, Kraken |
