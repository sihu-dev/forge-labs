## 4. AI 셀 함수

```
=AI(prompt)              → 자연어로 뭐든 생성
=AI_SUMMARY(range)       → 범위 요약
=AI_EXTRACT(file, field) → PDF에서 필드 추출
=AI_SCORE(bidId)         → 낙찰 확률 예측
=AI_PROPOSAL(bidId)      → 제안서 초안 생성
=AI_PRICE(bidId, margin) → 적정 가격 추천
```
