# HEPHAITOS B2B2C 사업 전략

> **핵심 전환**: B2C → B2B2C (유튜버 파트너십 모델)
> **법적 포지셔닝**: 교육 플랫폼 (투자 자문 아님)
> **작성일**: 2025-12-19

---

## 1. 사업 개요

### 1.1 원라인 피치

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   "증권/코인 유튜버를 위한 White-Label 교육 플랫폼"                     │
│                                                                         │
│   유튜버가 자신만의 브랜드로 트레이딩 교육 → 수강생이 AI 에이전트 생성  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 비즈니스 모델 구조

```

  수강생: 교육 수료 → AI 에이전트 빌드
```

### 1.3 각 주체별 가치

| 주체 | 얻는 것 | 제공하는 것 |
|------|--------|------------|
| **유튜버** | 유료 교육 플랫폼, 추가 수익 | 구독자 트래픽, 콘텐츠 |
| **수강생** | 체계적 교육, AI 에이전트 | 수강료, 크레딧 구매 |
| **HEPHAITOS** | 유저 획득 (CAC 절감), 매출 | 플랫폼 인프라, 기술 |

## 2. 유튜버 파트너 프로그램

### 2.1 타겟 유튜버 프로필

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        타겟 유튜버 세그먼트                              │
└─────────────────────────────────────────────────────────────────────────┘

  Tier 1: 메가 인플루언서 (구독자 50만+)
  ──────────────────────────────────────
  • 슈카월드, 삼프로TV, 신사임당
  • 이미 유료 강의 운영 중
  • 제안: 전용 화이트라벨 + 프리미엄 수수료

  Tier 2: 매크로 인플루언서 (구독자 10-50만)
  ──────────────────────────────────────────
  • 주식/코인 전문 채널 다수
  • 유료화 고민 중이나 플랫폼 없음
  • 제안: 표준 파트너십 + 수익 쉐어

  Tier 3: 마이크로 인플루언서 (구독자 1-10만)
  ──────────────────────────────────────────
  • 전문성 있으나 인프라 부족
  • 가장 적극적인 얼리어답터
  • 제안: 셀프서비스 + 기본 수수료

  초기 집중: Tier 2-3 (20-30개 채널 확보 목표)
```

## 4. 법적 우회 전략

### 4.1 포지셔닝

```
┌─────────────────────────────────────────────────────────────────────────┐
│                       법적 포지셔닝 전략                                 │
└─────────────────────────────────────────────────────────────────────────┘

  ❌ 우리가 아닌 것:
  ─────────────────
  • 투자 자문업자 (금융위 등록 필요)
  • 투자 일임업자
  • 자산운용사
  • 로보어드바이저

  ✅ 우리가 맞는 것:
  ─────────────────
  • 교육 콘텐츠 플랫폼 (클래스101, 인프런과 동일)
  • 소프트웨어 도구 제공자 (백테스트 도구)
  • 커뮤니티 플랫폼

  핵심 차이:
  ──────────
  "우리는 전략을 추천하지 않는다"
  "사용자가 스스로 학습하고 만드는 것을 돕는다"
  "교육자(유튜버)가 콘텐츠를 제공하고, 우리는 도구만 제공"
```

### 4.2 필수 면책 조항

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         면책 조항 (필수 표시)                            │
└─────────────────────────────────────────────────────────────────────────┘

  회원가입 시:
  ───────────
  □ 본 서비스는 교육 목적으로만 제공됩니다.
  □ 투자 조언, 추천, 또는 권유가 아닙니다.
  □ 모든 투자 결정과 그에 따른 손익은 회원 본인의 책임입니다.
  □ 과거 백테스트 결과는 미래 수익을 보장하지 않습니다.

  에이전트 생성 시:
  ─────────────────
  ⚠️ 경고: 자동 매매는 원금 손실 위험이 있습니다.
  □ 본인이 직접 설계한 전략임을 확인합니다.
  □ 손실 가능성을 이해하고 감수합니다.
  □ 투자 가능 금액만 사용합니다.

  실전 연결 시:
  ─────────────
  ⚠️ 최종 확인: 실제 자금이 거래됩니다.
  □ 증권사 API 연결은 본인 책임입니다.
  □ 시스템 오류로 인한 손실도 본인 책임입니다.
  □ HEPHAITOS는 투자 결과에 대해 어떠한 책임도 지지 않습니다.
```

### 4.3 법적 구조

```
  유튜버 (교육자)
       │
       │  "교육 콘텐츠 제작자 계약"
       │  → 유튜버가 교육 콘텐츠 책임
       │
       ▼
  HEPHAITOS (플랫폼)
       │
       │  "소프트웨어 서비스 이용약관"
       │  → 도구 제공만, 투자 조언 아님
       │
       ▼
  수강생 (사용자)
       │
       │  "교육 수강 및 도구 사용 동의"
       │  → 모든 투자 결정 본인 책임
       │
       ▼
  증권사 (KIS, Alpaca)
       │
       │  사용자 직접 계약
       │  → HEPHAITOS는 중개하지 않음
```

## 5. 제품 형태 옵션

### 5.1 옵션 비교

```
┌─────────────────────────────────────────────────────────────────────────┐
│                      제품 제공 형태 옵션                                 │
└─────────────────────────────────────────────────────────────────────────┘

  Option A: 통합 플랫폼 (권장 ✅)
  ───────────────────────────────
  • 유튜버 전용 서브도메인 제공
  • 예: trade.주식왕.com
  • 화이트라벨 브랜딩
  • 모든 기능 통합

  장점: 관리 용이, 일관된 경험
  단점: 유튜버 자유도 낮음


  Option B: API 제공
  ──────────────────
  • 에이전트 빌더 API
  • 백테스트 API
  • 유튜버가 자체 프론트엔드 구축

  장점: 유튜버 자유도 높음
  단점: 기술력 필요, 지원 비용 높음


  Option C: 크레딧 리셀러
  ───────────────────────
  • 유튜버가 크레딧을 대량 구매
  • 수강생에게 재판매
  • 가격 결정권 유튜버에게

  장점: 단순한 구조
  단점: 플랫폼 접점 약함


  권장: Option A + Option C 하이브리드
  ─────────────────────────────────────
  • 기본은 통합 플랫폼
  • 크레딧은 유튜버가 번들 판매 가능
```
