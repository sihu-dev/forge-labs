```

│ - 투자자 성향 분석 자동화                                    │
```

#### 대응 전략

| 옵션 | 설명 | 실현 가능성 |
|------|------|-----------|
| **A. 교육 플랫폼 포지셔닝** | "투자 교육/시뮬레이션"으로 명확히 구분, 실제 자금 거래 없음 | 🟢 높음 |
| **B. 금융규제 샌드박스** | FSC 혁신금융서비스 지정 신청 | 🟡 중간 |
| **C. 투자일임업 등록** | 15억원 자본금 + 인력 요건 충족 | 🔴 낮음 (초기 단계) |

#### 권장 접근법

**"Copy → Learn → Build" 파이프라인을 다음과 같이 수정:**

1. **Copy**: 가상 포트폴리오로 카피 (실제 자금 X) → 규제 회피
2. **Learn**: 전략 분석/교육 콘텐츠 → 교육 플랫폼
3. **Build**: 백테스팅/시뮬레이션 → 실제 거래는 사용자가 본인 증권사 연동

**결론**: 교육/시뮬레이션 포지셔닝 유지 시 규제 리스크 관리 가능
