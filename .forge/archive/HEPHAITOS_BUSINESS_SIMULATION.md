## 1. 프로젝트 자산 현황

### 기존 프로젝트 병합 대상

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         현재 보유 자산                                   │
└─────────────────────────────────────────────────────────────────────────┘

  HEPHAITOS (메인)                    명량 MVP (보류)
  ──────────────────                  ──────────────────
  📁 436개 TypeScript 파일            📁 다른 도메인으로 전환됨
  ✅ Exchange 통합 (KIS, Binance)     ⚠️ LawTalk AI로 변경
  ✅ Broker 어댑터                    → 트레이딩 자산 없음
  ✅ AI 전략 생성
  ✅ 백테스팅 엔진
  ✅ 결제 시스템 (Toss)
  ✅ 관리자 대시보드
  ✅ 셀럽 미러링 기능
```

```
┌─────────────────────────────────────────────────────────────────────────┐

│  → HEPHAITOS 단독으로 FORGE TRADING 구축                                │
│  → 전략 시뮬레이터는 HEPHAITOS 내 신규 개발                              │
└─────────────────────────────────────────────────────────────────────────┘

## 2. 사업 컨셉 정의

### 2.1 원라인 피치

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   "트레이딩을 게임처럼, 성장을 레벨업처럼"                               │
│                                                                         │
│   고수 따라하기 → 원리 이해 → 나만의 전략 만들기                        │
│   Copy → Learn → Build                                                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.2 핵심 가치 제안 (Value Proposition)

| 사용자 Pain Point | HEPHAITOS 솔루션 |
|------------------|-----------------|
| "트레이딩 어디서 시작해야 할지 모르겠어" | 검증된 고수 전략 1-클릭 복사 |
| "왜 이게 수익나는지 이해 안 돼" | AI 튜터 + 시각화 백테스트 |
| "내 전략 만들고 싶은데 코딩 못해" | No-Code 전략 빌더 |
| "수익률 검증하기 어려워" | 실시간 시뮬레이션 + 과거 데이터 |

### 2.3 타겟 사용자

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         타겟 페르소나                                    │
└─────────────────────────────────────────────────────────────────────────┘

  Primary: 2030 초보 투자자
  ─────────────────────────
  • 주식/코인 관심 있지만 시작 못 함
  • 유튜브로 공부하다 포기
  • 소액(100만원 이하)으로 시작 희망
  • "잃지 않는 투자"에 관심

  Secondary: 3040 직장인 투자자
  ─────────────────────────────
  • 시간 없어서 자동화 원함
  • 어느 정도 경험 있음
  • 전략 다각화 니즈
  • 월 10-50만원 지불 의사

  Tertiary: 전업 트레이더
  ────────────────────────
  • 자신의 전략 판매 희망
  • 고급 백테스트 도구 필요
  • 커뮤니티 영향력 구축
```

## 6. 리스크 및 완화

### 6.1 리스크 매트릭스

| 리스크 | 확률 | 영향도 | 완화 전략 |
|--------|------|--------|----------|
| **금융 규제** | 중 | 고 | "교육 플랫폼" 포지셔닝, 투자 조언 아님 면책 |
| **대형사 진입** | 중 | 중 | 빠른 커뮤니티 구축, 니치 집중 |
| **기술 장애** | 저 | 고 | 모니터링 + 장애 알림 시스템 |
| **고수 이탈** | 중 | 중 | 수익 쉐어 + 인센티브 프로그램 |
| **사용자 손실 클레임** | 고 | 중 | 시뮬레이션 강조, 면책조항, 손실 한도 설정 |

### 6.2 법률 준수 체크리스트

```
□ 투자 조언업 등록 여부 확인
