# 멘토-멘티 협업 트레이딩 빌드 시나리오

> **목적**: 멘토와 멘티가 함께 전략을 빌드할 때의 최적 경우의 수
> **작성일**: 2025-12-19

---

## 1. 협업 빌드 모드 개요

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    협업 빌드 모드 (COLLAB BUILD MODE)                     │
└─────────────────────────────────────────────────────────────────────────┘

  멘토 역할                              멘티 역할
  ────────                              ────────
  • 전략 템플릿 제공                     • 템플릿 커스터마이징
  • 실시간 가이드                        • 파라미터 조정
  • 코드 리뷰/피드백                     • 백테스트 실행
  • 베스트 프랙티스 공유                 • 질문/피드백 요청

              ┌─────────────────────────────┐
              │      실시간 협업 세션        │
              │                             │
              │  멘토 ◄────────────► 멘티   │
              │        WebSocket           │
              │                             │
              │  [공유 캔버스] [채팅] [음성] │
              └─────────────────────────────┘
```

## 2. 협업 시나리오 매트릭스

### 2.1 시나리오 분류

| # | 시나리오 | 멘토 참여도 | 멘티 숙련도 | 난이도 |
|---|---------|-----------|-----------|-------|
| A | 완전 가이드 | 100% | 초급 | ⭐ |
| B | 부분 가이드 | 50% | 중급 | ⭐⭐ |
| C | 리뷰 전용 | 20% | 중급 | ⭐⭐ |
| D | 자유 빌드 | 0% (비동기) | 고급 | ⭐⭐⭐ |
| E | 라이브 코딩 | 80% | 혼합 | ⭐⭐ |
| F | 1:N 그룹 빌드 | 60% | 혼합 | ⭐⭐⭐ |

## 3. 시나리오 A: 완전 가이드 모드

```
┌─────────────────────────────────────────────────────────────────────────┐
│  시나리오 A: 완전 가이드 (초급 멘티용)                                    │
└─────────────────────────────────────────────────────────────────────────┘

  상황: 처음 에이전트를 만드는 멘티에게 멘토가 1:1로 가이드

  Step 1: 세션 시작
  ─────────────────
  ┌─────────────────────────────────────────────────────────────────────┐
  │  📹 라이브 빌드 세션                                                 │
  │                                                                     │
  │  ┌─────────────────────────────┬─────────────────────────────────┐ │
  │  │  멘토 화면 (공유 중)         │  내 작업 화면                   │ │
  │  │                             │                                 │ │
  │  │  [RSI 전략 빌더]            │  [따라하기 모드 ON]             │ │
  │  │                             │                                 │ │
  │  │  "여기서 RSI 기간을         │  RSI 기간: [14] ← 입력 대기    │ │
  │  │   14로 설정합니다"          │                                 │ │
  │  │                             │                                 │ │
  │  └─────────────────────────────┴─────────────────────────────────┘ │
  │                                                                     │
  │  💬 채팅: "선생님, 14 대신 7로 하면 어떻게 되나요?"                │
  └─────────────────────────────────────────────────────────────────────┘

  Step 2: 단계별 검증
  ───────────────────
  멘토 액션            시스템 검증              멘티 화면
  ──────────          ──────────              ──────────
  "RSI 14 설정"   →   멘티 입력 대기      →   [14] 입력 필드 하이라이트
       ↓                   ↓                        ↓
  멘티가 입력     →   값 검증 (동일?)     →   ✅ "잘 따라오셨어요!"
       ↓                   ↓                        ↓
  다음 단계       →   진도 자동 업데이트   →   진도 +10%

  Step 3: 실시간 피드백
  ─────────────────────
  ┌─────────────────────────────────────────────────────────────────────┐
  │  멘티가 다른 값 입력 시                                              │
  │                                                                     │
  │  멘티 입력: RSI 기간 = 7                                            │
  │                                                                     │
  │  시스템 알림: "멘토 추천값(14)과 다릅니다"                          │
  │                                                                     │
  │  [추천값 사용] [내 값 유지] [멘토에게 질문]                         │
  │                                                                     │
  │  → "멘토에게 질문" 선택 시                                          │
  │                                                                     │
  │  자동 질문 생성: "RSI 기간을 7로 설정해도 될까요?"                  │
  │                                                                     │
  │  멘토 답변: "7은 단기 트레이딩용입니다.                             │
  │            스윙이면 14, 데이트레이딩이면 7 추천"                    │
  └─────────────────────────────────────────────────────────────────────┘

  최종 결과:
  ──────────
  • 멘티: 첫 에이전트 완성 + 학습 기록 저장
  • 멘토: 가이드 완료 → 리뷰 요청 가능
  • 시스템: 협업 세션 리플레이 저장
```

## 4. 시나리오 B: 부분 가이드 모드

```
┌─────────────────────────────────────────────────────────────────────────┐
│  시나리오 B: 부분 가이드 (중급 멘티용)                                   │
└─────────────────────────────────────────────────────────────────────────┘

  상황: 기본기가 있는 멘티가 특정 부분만 질문

  플로우:
  ───────
  멘티 자율 빌드 → 막히는 부분 발생 → 멘토 호출 → 해당 부분만 가이드

  Step 1: 자율 빌드 중 막힘
  ─────────────────────────
  ┌─────────────────────────────────────────────────────────────────────┐
  │  내 에이전트: RSI + 볼린저밴드 복합 전략                            │
  │                                                                     │
  │  ✅ RSI 설정 완료                                                   │
  │  ✅ 볼린저밴드 설정 완료                                            │
  │  ⚠️ 복합 조건 설정 ← 막힘                                          │
  │                                                                     │
  │  "두 조건을 AND로 합치는 방법을 모르겠어요"                         │
  │                                                                     │
  │  [AI 도움말] [멘토에게 질문] [커뮤니티 검색]                        │
  └─────────────────────────────────────────────────────────────────────┘

  Step 2: 멘토 호출 (비동기 → 동기 전환)
  ─────────────────────────────────────
  멘티                     시스템                      멘토
  ────                     ─────                      ────
  "멘토에게 질문"     →   알림 발송              →   "수강생 도움 요청"
       ↓                      ↓                           ↓
  대기 (5분)          →   멘토 접속 확인         →   "지금 도와줄게요"
       ↓                      ↓                           ↓
  세션 연결           ←   WebSocket 연결         ←   화면 공유 시작

  Step 3: 포인트 가이드
  ─────────────────────
  ┌─────────────────────────────────────────────────────────────────────┐
  │  멘토가 특정 영역만 하이라이트                                       │
  │                                                                     │
  │  [멘토 화면]                                                        │
  │  ┌─────────────────────────────────────────────────────────────┐   │
  │  │  IF                                                         │   │
  │  │    RSI(14) < 30                                             │   │
  │  │  ┌─────────────────────────────────────────────────────┐   │   │
  │  │  │ AND  ← 여기를 클릭하세요                            │   │   │
  │  │  │      볼린저밴드 하단 터치                           │   │   │
  │  │  └─────────────────────────────────────────────────────┘   │   │
  │  │  THEN                                                       │   │
  │  │    매수                                                     │   │
  │  └─────────────────────────────────────────────────────────────┘   │
  │                                                                     │
  │  💬 "AND 블록을 드래그해서 두 조건 사이에 놓으세요"                │
  └─────────────────────────────────────────────────────────────────────┘

  Step 4: 완료 후 분리
  ────────────────────
  • 멘티: 해당 부분 이해 → 자율 빌드 계속
  • 멘토: 세션 종료 → 다음 호출 대기
  • 기록: 도움 받은 영역 태깅 → 향후 복습 추천
```

## 5. 시나리오 C: 리뷰 전용 모드

```
┌─────────────────────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────────────────────┘

  상황: 멘티가 에이전트를 완성한 후 멘토에게 리뷰 요청

  Step 1: 리뷰 요청
  ─────────────────
  ┌─────────────────────────────────────────────────────────────────────┐
  │  📋 에이전트 리뷰 요청                                              │
  │                                                                     │
  │  에이전트명: RSI_역추세_v3                                          │
  │  백테스트 결과: +18.2% (6개월)                                      │
  │  MDD: -8.4%                                                         │
  │                                                                     │
  │  요청 메모:                                                         │
  │  "손절 로직이 적절한지 확인 부탁드립니다"                           │
  │                                                                     │
  │  [리뷰 요청 제출]                                                   │
  └─────────────────────────────────────────────────────────────────────┘

  Step 2: 멘토 리뷰 (비동기)
  ─────────────────────────
  ┌─────────────────────────────────────────────────────────────────────┐
  │  📝 리뷰 작성 (멘토 화면)                                           │
  │                                                                     │
  │  [전략 코드]                     [리뷰 코멘트]                      │
  │  ┌─────────────────────────┐    ┌─────────────────────────────┐   │
  │  │ IF RSI < 30             │    │                             │   │
  │  │   AND 거래량 > 150%     │    │ ✅ 진입 조건 OK             │   │

  │  │                         │    │                             │   │

  │  │ THEN 손절         ★    │    │    -7%~-8% 권장             │   │
  │  │                         │    │                             │   │
  │  │ IF 손익 > 10%           │    │ ✅ 익절 적정                │   │
  │  │ THEN 익절               │    │                             │   │
  │  └─────────────────────────┘    └─────────────────────────────┘   │
  │                                                                     │
  │  종합 평가: ⭐⭐⭐⭐ (4/5)                                          │
  │                                                                     │
  │  [리뷰 완료] [영상 피드백 추가]                                     │
  └─────────────────────────────────────────────────────────────────────┘

  Step 3: 리뷰 수신
  ─────────────────
  멘티에게 알림 → 리뷰 확인 → 수정 적용 → 재백테스트

  리뷰 상태 흐름:
  REQUESTED → REVIEWING → REVIEWED → APPLIED
       ↓           ↓          ↓          ↓
    요청완료    멘토검토중   피드백완료  멘티적용완료
```

```
  │  │                                   │                           │ │
```

## 7. 시나리오 F: 1:N 그룹 빌드

```
┌─────────────────────────────────────────────────────────────────────────┐

  그룹 빌드 구조:
  ───────────────
  ┌─────────────────────────────────────────────────────────────────────┐

  │                                                                     │

  │  │  참여자 현황 (8/10명 접속)                                  │   │
  │  │                                                             │   │
  │  │  김철수 ████████░░ 80% [Step 4/5]                          │   │
  │  │  이영희 ██████░░░░ 60% [Step 3/5] ⚠️ 막힘                  │   │
  │  │  박민수 ██████████ 100% [완료] ✅                          │   │
  │  │  정다운 ████░░░░░░ 40% [Step 2/5]                          │   │
  │  │  ...                                                        │   │
  │  └─────────────────────────────────────────────────────────────┘   │
  │                                                                     │
  │  [전체 공지] [이영희 도움주기] [박민수 리뷰]                        │
  └─────────────────────────────────────────────────────────────────────┘

  그룹 빌드 기능:
  ───────────────
  1. 동시 진도 관리
     • 전체 진행 브로드캐스트
     • 개인별 진도 추적

  2. 막힘 자동 감지
     • 5분 이상 같은 단계 → 알림
     • 멘토 개입 제안

  3. 빠른 학생 활용
     • 완료자 → 보조 멘토 역할
     • 피어 헬프 시스템

  4. 그룹 Q&A
     • 공통 질문 → 전체 답변
     • 개인 질문 → 1:1 채팅

  그룹 빌드 타임라인:
  ───────────────────
  00:00  세션 시작, 오늘 목표 설명
  00:10  Step 1 시연 → 전원 따라하기
  00:20  Step 1 완료 확인 (80% 이상)
  00:25  Step 2 시연 → 따라하기
  ...
  01:30  전원 완료 → 백테스트 비교
  01:45  우수 전략 공유 → 세션 종료
```
