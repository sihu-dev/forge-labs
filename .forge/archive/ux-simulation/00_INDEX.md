## 📋 문서별 핵심 내용

### 1. 멘토-멘티 협업 빌드 시나리오 (01)

| 시나리오 | 멘토 참여도 | 대상 멘티 | 핵심 기능 |
|---------|-----------|---------|----------|
| A. 완전 가이드 | 100% | 초급 | 실시간 따라하기, 단계별 검증 |
| B. 부분 가이드 | 50% | 중급 | 막힘 시 호출, 포인트 가이드 |
| C. 리뷰 전용 | 20% | 중급 | 비동기 리뷰, 피드백 제공 |
| D. 자유 빌드 | 0% | 고급 | 완전 자율, 필요시 질문 |
| E. 라이브 코딩 | 80% | 혼합 | YouTube Live 연동, 대규모 |
| F. 그룹 빌드 | 60% | 혼합 | 5-10명 소규모 워크숍 |

**핵심 결정 트리**:
```
첫 에이전트? → Yes → 시나리오 A (완전 가이드)
            → No → 막힘? → Yes → 시나리오 B (부분 가이드)
                        → No → 완성? → Yes → 시나리오 C (리뷰)
                                      → No → 계속 빌드
```

### 2. No-Code 빌더 UI (02)

**레이아웃 구조**:
```
┌─────────────────────────────────────────────────────────────┐
│  블록 팔레트 (170px) │ 캔버스 (flex-1) │ 설정 패널 (280px) │
├─────────────────────────────────────────────────────────────┤
│                     미니 차트 영역                           │
└─────────────────────────────────────────────────────────────┘
```

**블록 카테고리**:
| 카테고리 | 블록 예시 | 색상 |
|---------|---------|------|
| 지표 | RSI, MACD, 볼린저밴드, MA | 파랑 |
| 조건 | 크다, 작다, 상향돌파, 하향돌파 | 보라 |
| 논리 | AND, OR, NOT | 노랑 |
| 액션 | 매수, 매도, 홀드 | 초록/빨강 |
| 리스크 | 손절, 익절, 포지션 크기 | 주황 |

**핵심 인터랙션**:
- 드래그 앤 드롭 블록 배치
- 포트 연결 (출력 → 입력)
- 실시간 값 표시
- 블록 상태 인디케이터

### 3. 백테스트 결과 해석 (03)

**핵심 지표 카드**:
| 지표 | 해석 | 기준 |
|-----|-----|-----|
| 총수익률 | 투자금 대비 수익 | 벤치마크 대비 비교 |
| 승률 | 수익 거래 비율 | 60% 이상 양호 |
| MDD | 최대 낙폭 | -10% 이내 양호 |
| 샤프지수 | 위험 대비 수익 | 1 이상 양호 |

**초보자 가이드 모드**:
- 종합 점수 (100점 만점)
- "쉽게 말해서..." 섹션
- 1,000만원 기준 예시
- 면책 고지 필수 표시

**AI 개선 제안**:
- 손절 타이밍 조정
- 진입 필터 추가
- 포지션 크기 최적화

### 4. 실계좌 연결 및 운용 (04)

**지원 브로커/거래소**:
| 구분 | 플랫폼 | 인증 방식 | 난이도 |
|-----|-------|---------|-------|
| 국내주식 | 한국투자증권 | API Key → Token | 중 |
| 해외주식 | Alpaca | OAuth 2.0 | 쉬움 |
| 코인 | Binance | API Key + HMAC | 중 |
| 코인 | 업비트 | API Key + JWT | 중 |

**안전장치**:
- 일일 손실 한도 설정
- 연속 손실 N회 자동 중지
- API 오류 감지
- 시장 급변 감지
- 비상 정지 버튼

**알림 시스템**:
- 거래 알림 (매수/매도/체결)
- 리스크 알림 (손실 한도)
- 성과 알림 (일간/주간 리포트)

## 🎯 전체 사용자 여정 (User Journey)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         HEPHAITOS 사용자 여정                            │
└─────────────────────────────────────────────────────────────────────────┘

  [멘티 여정]

  1. 발견 (Discovery)
     YouTube 영상 → 랜딩 페이지 → 멘토 채널
                                    │
  2. 가입 (Onboarding)              ▼
     회원가입 → 역할 선택 → 프로필 설정
                              │
  3. 학습 (Learning)              ▼
     강의 탐색 → 결제 → 수강 → 진도 관리 → Q&A
                                     │
  4. 빌드 (Building)                 ▼
     템플릿 선택 → No-Code 빌더 → 커스터마이징 → 백테스트
                                              │
  5. 운용 (Operating)                         ▼
     계좌 연결 → 에이전트 배포 → 실시간 모니터링 → 성과 리포트
                                                   │
  6. 성장 (Growing)                                ▼
     멘토 피드백 → 전략 개선 → 새 전략 학습 → [반복]


  [멘토 여정]

  1. 파트너십 (Partnership)
     파트너 신청 → 심사 → 승인
                      │
  2. 콘텐츠 제작 (Content Creation)
     강의 생성 → 커리큘럼 설계 → 템플릿 제공
                              │
  3. 교육 (Teaching)            ▼
     수강생 관리 → Q&A 답변 → 협업 빌드 세션
                                 │
  4. 수익화 (Monetization)       ▼
     매출 발생 → 정산 (70:30) → 수익 분석
```
